# BD1ToOBJ

BD1形式のモデルをOBJ形式に変換します。

## 使い方

### オプション一覧

| オプション           | 意味                      |
| :------------------- | :------------------------ |
| -i, --inputFilepath  | 入力するBD1のファイルパス |
| -o, --outputFilepath | 出力するOBJのファイルパス |
| -z, --invertZ        | Z軸を反転                 |
| -f, --flipV          | UVのV座標を反転           |
| -h, --help           | ヘルプを表示              |
| -v, --version        | バージョン情報を表示      |

### 使用例

```
java -jar bd1-to-obj-1.0.1.jar -i map.bd1 -o map.obj -z -f
```
